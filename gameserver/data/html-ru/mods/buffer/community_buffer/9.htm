Profile Instruction

1) Creating a profile To create a profile, you need to click on the “Start Recording” button. Next, you apply the necessary spells for yourself at the NPC Buffer, after the set of buffs is completed, click on the “Save” button and select a profile number from 1st to 4th.

2) Using a saved profile After you have created the Profile, to use the created profile you just need to click on the profile in which you saved your set of spells - it will automatically overlap with you.

3) Overwriting a saved profile To rewrite your profile - you need not to have spells on your character, click "Start recording" and then "Save". After these actions, the profile is deleted.
